# Linguagem de Programação - ADS (FATEC)

Repositório da disciplina de Linguagem de Programação do curso de Análise e Desenvolvimento de Sistemas.

**Tecnologias:** TypeScript, Node.js, Express

**Ferramentas:** VS Code, Git/GitHub, GitHub Copilot

---

## Ementa

Variáveis, constantes, operadores e expressões. Comandos de desvio. Controle de malhas. Vetores e ponteiros. Funções de biblioteca. Estruturas, uniões e tipos definidos pelo usuário. Manipulação de arquivos.

## Objetivo

Solucionar problemas utilizando a lógica de programação e a implementação de programas por meio de uma linguagem de programação.

## Como usar este repositório

```bash
# Clone o repositório
git clone <url-do-repositorio>

# Entre na pasta do projeto
cd AulasLingProgADS

# Navegue até a aula desejada
cd aula03-typescript-variaveis-tipos

# Instale as dependências (quando houver package.json)
npm install
```

## Pré-requisitos

- [Node.js](https://nodejs.org/) (versão LTS)
- [VS Code](https://code.visualstudio.com/)
- [Git](https://git-scm.com/)
- [GitHub Copilot](https://github.com/features/copilot) (extensão do VS Code)
