# Linguagem de Programacao - ADS (FATEC)

Repositorio da disciplina de Linguagem de Programacao do curso de Analise e Desenvolvimento de Sistemas.

**Tecnologias:** TypeScript, Node.js, Express
**Ferramentas:** VS Code, Git/GitHub, GitHub Copilot

---

## Ementa

Variaveis, constantes, operadores e expressoes. Comando de desvio. Controle de malhas. Vetores e ponteiros. Funcoes de biblioteca. Estruturas, unioes e tipos definidos pelo usuario. Manipulacao de arquivos.

## Objetivo

Solucionar problemas utilizando a logica de programacao e a implementacao de programas por meio de uma linguagem de programacao.

## Como usar este repositorio

```bash
# Clone o repositorio
git clone <url-do-repositorio>

# Entre na pasta do projeto
cd AulasLingProgADS

# Navegue ate a aula desejada
cd aula03-typescript-variaveis-tipos

# Instale as dependencias (quando houver package.json)
npm install
```

- [Node.js](https://nodejs.org/) (versao LTS)
